# Uber Trip Analysis Project 📊

This project was completed as part of my **Unified Mentor Internship Program**.
The goal of this project is to analyze Uber trip data and build an interactive **Power BI dashboard** to uncover trends, patterns, and operational insights.

---

🚀 Project Overview

The dashboard provides a clear visualization of Uber trip activity, vehicle availability, and dispatch base performance.
It helps understand demand patterns and operational efficiency using interactive filters and visual insights.

---

📌 Key Highlights

• Total Trips Analyzed: **4 Million+**
• Total Active Vehicles: **463K**
• Average Trips Per Day: **11.67K**
• Maximum Daily Trips: **46K**
• Minimum Daily Trips: **629**

---

📊 Dashboard Insights

The dashboard includes the following visual analysis:

• Daily Uber Trip Trend
• Active Vehicles Trend
• Demand vs Supply Trend (Trips vs Vehicles)
• Trips by Dispatching Base
• Interactive Date and Month Filters

These insights help understand ride demand, vehicle supply, and operational patterns.
